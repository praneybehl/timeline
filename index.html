Timeline

timeline component with tags for video players
download .ZIPdownload .TGZ

Small demo


    $('#timeline').timeline({
        duration: 60 * 2 + 25,
        framesNumber: 10,
        width: 470,
        tags: [
            {title: 'One', line: 0, time: 0, length: 15},
            {title: 'Two', line: 0, time: 30, length: 15},
            {title: 'Three', line: 1, time: 35, length: 20},
            {title: 'Four', line: 2, time: 40, length: 50}
        ]
    })

You can drug'n'drop and resize the tags
