Timeline

timeline component with tags for video players
download .ZIPdownload .TGZ

Small demo

You can drag'n'drop and resize the tags


    var timeline = new Timeline('#timeline', {
        width: 470,
        duration: 60 * 2 + 25,
        framesNumber: 10,
        tags: [
            {title: 'One', length: 15},
            {title: 'Two', start: 30, length: 15},
            {title: 'Three', start: 35, length: 20},
            {title: 'Four', start: 40, length: 50}
        ]
    });

With custom frames order and without align to top and disabled tag "One"


    var timeline = new Timeline('#timeline2', {
        width: 470,
        duration: 60 * 2 + 25,
        framesNumber: 15,
        framesOrder: ['.main', '.half'],
        align: false,
        tags: [
            {title: 'One', line: 1, length: 15, disabled: true},
            {title: 'Two', start: 30, length: 15},
            {title: 'Three', line: 3, start: 35, length: 20},
            {title: 'Four', line: 3, start: 65, length: 15}
        ]
    });
